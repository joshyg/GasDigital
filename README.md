# gas-digital-rn

## getting started
* install node, xcode, etc.
* npm install
* install react-native-debugger
* install react-native-cli if not installed

## running
* open react native debugger (its an app)
* ios: 
  * `react-native run-ios`
* android:
  * open android studio
  * start an emulator
  * close android stuiod
  * `react-native run-android`
